## 我为什么喜欢编程？

程序是一种能让人体会到即刻兑现努力的东西，这是它最迷人的地方。包括游戏，也是一种程序。在程序里面，1+1就是等于2，凡事都是确定的。

相比现实世界：
- 追了很久都没有追到的女神；
- 考了很多次都没有考过的CPA；
- 找了很久都没有找到的工作

程序给你带来的，是让你即刻体会到成就感的快感。抛开编程水平不谈，你写一行程序，就会有一行的效果，马上见效。__编程，应该是唯一一个普通人可以使用的魔法了。__
